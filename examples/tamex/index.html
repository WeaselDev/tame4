Examples Using the TAME Library for the Beckhoff ADS-Webservice

  • Basic Example

    This is an example for writing and reading single variables.
    Cyclic polling is also included.

  • Handles

    Almost the same as the Basic Example, but with other variables and using handles to access them.

  • Arrays

    An example for reading and writing arrays.

  • Structures

    A demo how to make live easier using sructures.

  • Array of Structures

    Read or write an array of structures with one request.
    I've already used it for fetching arrays with more than
    2000 values.

  • Example Using Ext JS 4

    I'm new to Ext and just played around with some demos provided
    by the developers. But I think the store system of Ext suites
    quite well with TAME, especially using structures. But see for
    yourself. This applies also to Sencha Touch, so it's worth having
    a look at if you plan developing an application for mobile devices.
    Visit www.sencha.com for licensing options if you want to use
    the framework.

  • Example Using SVG

    Example showing SVG graphics with the vex.js library for input.
    SVG created in Inkscape with ID's and exported as "Optimized SVG"
